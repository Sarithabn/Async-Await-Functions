
# Async Await User API

A simple web application that fetches user data from the JSONPlaceholder API using async/await syntax.

## Features
* Fetches user data from the JSONPlaceholder API
* Displays user details on the page
* Loading indicator while fetching data
* Basic styling with CSS

## Tech Stack
* HTML5
* CSS3
* JavaScript (ES6+)

## Usage
1. Open `index.html` in a web browser
2. Click the "Fetch User" button to retrieve user data
3. User details will be displayed on the page

## Code Structure
* `index.html`: HTML structure for the application
* `style.css`: CSS styles for the application
* `script.js`: JavaScript code for fetching and displaying user data

## API Endpoint
* `https://jsonplaceholder.typicode.com/users/1`: Fetches user data for user ID 1
